# AI Landmark Detection System

An AI-powered computer vision project that detects and classifies famous landmarks from images using Deep Learning and TensorFlow.

## Features

- Landmark image classification
- Deep learning based prediction
- Image preprocessing and augmentation
- Model training and evaluation
- Real-time image testing
- Easy deployment and scalability

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn
